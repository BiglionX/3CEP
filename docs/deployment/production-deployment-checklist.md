## 🚀 部署执行步骤

### 1. 代码部署

```bash
# 拉取最新代码
git pull origin main

# 安装依赖
npm ci

# 构建项目
npm run build
```

### 2. 环境配置

```bash
# 设置环境变量
export DEEPSEEK_API_KEY="your_deepseek_key"
export QWEN_API_KEY="your_qwen_key"

# 或在Windows中设置
set DEEPSEEK_API_KEY=your_deepseek_key
set QWEN_API_KEY=your_qwen_key
```

### 3. 服务启动

```bash
# 启动生产服务
npm run start

# 或使用PM2
pm2 start ecosystem.config.js
```

### 4. 健康检查

```bash
# 检查服务状态
curl http://localhost:3001/api/b2b-procurement/parse-demand-llm

# 运行健康测试
node scripts/test-large-model-integration.js
```
