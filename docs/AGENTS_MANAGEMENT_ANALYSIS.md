## 🎯 核心概念解析

### 1. 智能体 (Agent)

**定义**: 平台中的服务单元，可以是 n8n 工作流或独立服务

**类型**:

- **n8n 型** - 基于 n8n 的工作流
- **service 型** - 独立的 HTTP 服务

**注册信息**:

```typescript
interface AgentRegistration {
  id: string; // 唯一标识
  name: string; // 名称
  domain: string; // 领域（如订单、采购等）
  type: 'n8n' | 'service'; // 类型
  endpoint: string; // 服务端点
  version: string; // 版本号
  description?: string; // 描述
  metadata: {
    latency_sensitive: boolean; // 延迟敏感性
    security_level: 'low' | 'medium' | 'high';
    traffic_level: 'low' | 'medium' | 'high';
    status_complexity: 'low' | 'medium' | 'high';
  };
  health_check_endpoint?: string; // 健康检查端点
  supported_operations: string[]; // 支持的操作列表
}
```

---

### 2. 智能体模板 (Agent Templates)

**页面**: `/admin/agent-templates`

**用途**: 预定义的智能体配置模板

**关系**:

```
智能体模板 → 创建智能体时选择 → 实例化为具体智能体
```

**类比**:

- 智能体模板 = 类（Class）
- 智能体 = 对象（Object）

---

### 3. 执行工作流 (Execute Workflow)

**页面**: `/admin/agents/execute`

**用途**: 手动触发智能体的执行流程

**关系**:

```
智能体注册 → 配置完成 → 执行工作流 → 产生执行记录
```

**功能**:

- ✅ 手动触发智能体
- ✅ 传入参数测试
- ✅ 查看执行结果

---

### 4. 模板库 (Templates)

**页面**: `/admin/agents/templates`

**用途**: n8n 的模板库，用于导入工作流模板

**关系**:

```
n8n 模板库 → 导入模板 → 创建工作流 → 注册为智能体
```

**数据来源**: n8n API (`${N8N_BASE_URL}/api/v1/templates`)

**功能**:

- ✅ 浏览 n8n 模板
- ✅ 按分类筛选
- ✅ 导入模板到 n8n
- ✅ 收藏常用模板

---

### 5. 环境变量 (Environment Variables)

**页面**: `/admin/agents/environment`

**用途**: 管理全局环境变量和密钥

**关系**:

```
环境变量 → 工作流/智能体使用 → 运行时读取配置
```

**变量类型**:

- **公开变量** - 普通配置项
- **密钥** - 敏感信息（加密存储）

**使用场景**:

- API 密钥
- 数据库连接字符串
- 第三方服务凭证
- 全局配置参数
