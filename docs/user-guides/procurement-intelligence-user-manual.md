## 🎯 系统概述

采购智能体是FixCycle平台的智能化采购决策支持系统，通过AI算法和大数据分析，为企业提供：

- **智能供应商匹配**：基于多维度评估自动推荐最优供应商
- **风险预警监控**：实时监控供应商风险并提供预警
- **价格趋势分析**：分析市场价格走势，优化采购时机
- **合同条款建议**：智能生成专业化的合同条款
- **采购决策支持**：综合各种因素提供采购决策建议

## 📊 核心功能操作指南

### 1. 智能仪表板

#### 1.1 仪表板概览

```
┌─────────────────────────────────────────────┐
│  📊 采购智能体仪表板                        │
├─────────────────────────────────────────────┤
│  关键指标区     |  活动动态区     |  快捷操作区  │
│                 |               |             │
│  • 总供应商数   |  • 最新活动    |  • 创建需求  │
│  • 活跃供应商   |  • 风险预警    |  • 评估供应商 │
│  • 月度支出     |  • 优化建议    |  • 查看报告  │
│  • 节省率       |               |             │
└─────────────────────────────────────────────┘
```

#### 1.2 关键指标解读

| 指标       | 说明                   | 正常范围         |
| ---------- | ---------------------- | ---------------- |
| 总供应商数 | 系统管理的供应商总数   | 根据企业规模而定 |
| 活跃供应商 | 近期有交易的供应商数量 | ≥ 总数的70%      |
| 月度支出   | 当月采购总支出         | 根据预算设定     |
| 节省率     | 智能优化带来的成本节省 | 通常10-25%       |

### 2. 采购需求管理

#### 2.1 创建智能采购需求

**操作步骤**:

1. 点击仪表板上的"创建采购需求"按钮
2. 填写需求基本信息：
   - 商品类别（下拉选择）
   - 采购数量
   - 期望交付日期
   - 预算范围
   - 特殊要求说明

3. 系统自动分析并显示：
   - 📈 市场价格趋势图
   - 🎯 推荐采购时机
   - 💰 预估成本区间
   - ⚠️ 潜在风险提示

#### 2.2 查看采购建议

系统会生成以下建议：

- **最优供应商列表**：按匹配度排序的供应商推荐
- **采购时机建议**：基于价格趋势的最佳采购时间
- **风险评估报告**：各供应商的风险等级和详细分析
- **合同条款草案**：初步的合同条款建议

### 3. 供应商管理

#### 3.1 供应商画像查看

**路径**: 供应商管理 → 供应商列表 → 点击具体供应商

**画像内容包括**:

- **基础信息**：公司名称、注册地、规模等
- **能力评分**：质量、价格、交付、服务、创新能力
- **风险等级**：低/中/高风险标识
- **历史表现**：过往合作记录和评价
- **改进建议**：针对该供应商的发展建议

#### 3.2 供应商搜索筛选

**筛选条件**:

- 📊 能力评分范围（如：80-100分）
- 🌍 注册国家/地区
- 🏢 企业规模（小型/中型/大型）
- 🏷️ 行业领域
- ⭐ 认证资质（ISO、环保等）

### 4. 风险监控

#### 4.1 风险预警查看

**预警级别**:

- 🟢 **低风险**：正常运营，无需特别关注
- 🟡 **中风险**：需要关注，建议加强监控
- 🔴 **高风险**：立即关注，考虑备选方案
- ⚫ **严重风险**：紧急处理，暂停合作

#### 4.2 风险详情分析

点击风险预警可查看：

- 风险触发原因
- 风险发展趋势
- 影响程度评估
- 缓解措施建议
- 历史风险记录

### 5. 市场分析

#### 5.1 价格趋势查看

**功能特点**:

- 📈 实时价格走势图
- 🌍 多地区价格对比
- 🔮 未来价格预测
- 💡 采购时机提醒

#### 5.2 供需分析

系统提供：

- 当前市场供需状况
- 季节性变化规律
- 替代品价格走势
- 市场集中度分析

## ❓ 常见问题解答

### Q1: 如何提高供应商匹配准确度？

**A**: 可以通过以下方式优化：

- 完善供应商基础信息
- 定期更新供应商绩效数据
- 根据业务特点调整评估权重
- 提供详细的采购需求描述

### Q2: 风险预警如何处理？

**A**: 处理流程：

1. 查看风险详情和触发原因
2. 评估对当前采购的影响
3. 考虑启动应急预案
4. 与供应商沟通改善措施
5. 持续监控风险变化

### Q3: 系统响应慢怎么办？

**A**:

- 检查网络连接状况
- 尝试刷新页面
- 避开高峰期使用
- 联系技术支持团队

### Q4: 如何获得更好的价格建议？

**A**:

- 提供准确的需求时间和数量
- 关注系统的价格趋势分析
- 考虑分批采购策略
- 参考多个地区的市场价格
