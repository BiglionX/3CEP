## 🎯 核心功能一览

### 1. 用户统计面板

**功能**: 实时用户数据统计和可视化展示
**访问路径**: `/admin/users` (默认标签页)
**主要指标**:

- 用户总数、活跃用户数、新增用户数
- 角色分布饼图
- 状态分布柱状图
- 增长趋势折线图

### 2. 高级搜索功能

**功能**: 多维度用户搜索和筛选
**特色能力**:

- 关键词搜索 (邮箱、用户名、ID)
- 角色筛选 (管理员、审核员等)
- 状态筛选 (正常、封禁、暂停)
- 时间范围筛选 (注册时间)
- 活跃度筛选 (最后活跃时间)
- 复合条件组合查询
- 筛选器保存和复用

### 3. 行为分析系统

**功能**: 深度用户行为洞察
**分析维度**:

- **访问频率**: 日均/周均/月均访问统计
- **功能使用**: 功能采用率、使用热度排行
- **活跃模式**: 最活跃时段、会话时长分析
- **用户分群**: 基于行为特征的智能分组

### 4. 智能推荐引擎

**功能**: 个性化用户和内容推荐
**推荐类型**:

- **相似用户推荐**: 基于协同过滤算法
- **内容推荐**: 功能、模板、工作流推荐
- **智能分组**: AI驱动的动态用户分群

### 5. 自动化规则引擎

**功能**: 规则驱动的智能运维
**支持模式**:

- 手动触发
- 定时执行 (Cron表达式)
- 事件驱动
- 实时监控
  **动作类型**: 通知发送、状态更新、角色分配等8种

### 6. 机器学习预测

**功能**: 前瞻式智能预测
**预测能力**:

- 流失风险预测 (准确率88%+)
- 用户价值评估 (相关性0.92)
- 行为趋势分析
- 参与度预测

## 🔧 API使用示例

### 获取用户统计

```bash
curl -X GET "http://localhost:3001/api/admin/users/stats?period=30d" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### 记录用户行为

```bash
curl -X POST "http://localhost:3001/api/admin/users/behavior" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "user_id": "user-uuid",
    "behavior_type": "page_view",
    "page_url": "/admin/users",
    "timestamp": "2026-02-27T10:30:00Z"
  }'
```

### 获取智能推荐

```bash
curl -X GET "http://localhost:3001/api/admin/users/recommendations/user-uuid?type=similar_users" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

1. **数据加载缓慢**: 检查网络连接和数据库性能
