# 数据分析平台使用指南

## 系统概述

数据分析平台提供完整的数据收集、处理、分析和可视化功能，支持业务决策和运营优化。

## 核心功能

### 1. 数据指标体系

- 业务关键指标(KPI)设计
- 多维度数据分析
- 实时数据监控
- 历史趋势分析

### 2. 数据收集管道

- 实时数据流处理
- 多源数据集成
- 自动清洗和验证
- 批量处理支持

### 3. 分析报表系统

- 自定义报表模板
- 多种图表类型支持
- 交互式数据探索
- PDF/Excel导出功能

### 4. 商业智能看板

- KPI仪表板
- 趋势预测分析
- 竞争对手对比
- 风险评估矩阵

## 使用说明

### 数据收集管道

访问路径: `/data-pipeline`

主要功能:

- 数据点收集和批量导入
- 实时数据流监控
- 数据清洗和验证
- 存储和检索管理

API使用示例:

```bash
# 收集单个数据点
POST /api/data-pipeline
{
  "action": "collect",
  "data": {
    "metricName": "user_activity",
    "value": 150,
    "dimensions": {"device": "mobile"},
    "source": "web"
  }
}

# 批量收集数据
POST /api/data-pipeline
{
  "action": "collectBatch",
  "data": [
    {"metricName": "page_views", "value": 1200},
    {"metricName": "conversion_rate", "value": 3.5}
  ]
}
```

### 分析报表系统

访问路径: `/analytics/reports`

操作流程:

1. 选择预设报表模板
2. 配置时间范围和维度
3. 生成分析报表
4. 查看可视化图表
5. 导出报表文件

支持的图表类型:

- 折线图: 趋势分析
- 柱状图: 对比分析
- 饼图: 占比分析
- 散点图: 相关性分析

### 商业智能看板

访问路径: `/bi/dashboard`

核心模块:

- **KPI监控**: 关键业务指标实时展示
- **趋势分析**: 业务发展趋势预测
- **竞争力分析**: 市场地位和竞争优势
- **风险评估**: 业务风险识别和评估

## API接口参考

### 数据收集API

```
POST /api/data-pipeline
- collect: 收集单个数据点
- collectBatch: 批量收集数据
- flush: 立即刷新缓冲区
- cleanup: 清理过期数据
```

### 数据查询API

```
GET /api/data-pipeline
- ?action=aggregated: 获取聚合数据
- ?action=export: 导出原始数据
```

### 报表生成API

```
POST /api/analytics/reports
- generate: 生成分析报表
- export: 导出报表文件
```
