# ProdCycleAI 项目说明书 (v5.0 整合版)

## 🎯 项目概述

ProdCycleAI 是一个基于现代 Web 技术栈构建的一站式 3C 电子产品循环经济平台，致力于为用户提供专业的设备维修、配件采购、智能估价和创新众筹服务。本项目采用前后端分离架构，基于 JAMstack 理念设计，旨在打造高性能、可扩展的智能服务生态系统。

### 核心价值主张

- **一站式循环经济平台**：整合维修服务、配件商城、智能估价、创新众筹四大核心功能
- **智能化决策支持**：基于机器学习的配件价格预测、维修方案推荐和设备生命周期管理
- **生态协同效应**：与 lionfix 3C 零配件资料库深度集成，实现数据共享和业务协同
- **设备生命周期透明化**：通过 LIFE 系统为用户提供可信的设备历史记录，增强品牌信任感
- **数据驱动的服务优化**：基于设备全生命周期数据，实现精准的 AI 诊断和个性化服务推荐
- **创新商业模式**：通过 FCX 通证经济体系和众筹创新平台，构建可持续发展的商业生态
- **全球化贸易服务**：提供完整的进出口贸易数字化管理解决方案，支持多语言多币种业务
- **供应链一体化**：整合订单、物流、仓储、合作伙伴管理，实现端到端的供应链可视化

### 平台愿景

通过 Procyc 生态系统，构建一个自驱动的全球 3C 服务网络：

- 小工厂以极低成本获得覆盖多国的服务网点
- 维修店通过优质服务赚取 FCX 和 FCX2，形成品牌信誉
- 消费者获得透明可靠的维修体验和创新产品机会
- 平台通过 FCX 流转和配件交易实现盈利
- 贸易商享受高效的数字化进出口管理服务
- 供应链伙伴实现业务协同和价值共创

### 核心业务模块

1. **维修服务模块**：维修店搜索、预约下单、订单管理
2. **配件商城模块**：3C配件浏览、购买、订单跟踪
3. **众筹创新模块**：新品众筹、项目管理、资金募集
4. **FCX联盟模块**：代币质押、奖励分配、联盟排行
5. **外贸管理模块**：进出口订单、合作伙伴、物流跟踪
6. **企业管理模块**：智能体定制、B2B采购、数据分析

### 数据架构模式

ProdCycleAI 采用**直连现有数据库**的集成策略，与 lionfix 系统形成紧密的数据协作关系：

- **数据来源**：直接连接 lionfix 3C 零配件资料库的核心数据库
- **协作模式**：lionfix 提供设备、配件、供应商等基础数据，ProdCycleAI 构建前台业务应用
- **技术实现**：通过只读账号直接查询 lionfix 数据库，封装成内部 API 供前端使用
- **架构优势**：两个系统保持独立部署，通过数据库层实现高效数据共享

## 💰 商业模式与通证经济

### FixCycle 3.0 通证经济体系

#### 通证设计原则

1. **中心化稳定**：FCX 为平台中心化积分，严格锚定法币（1 FCX = 0.01 USD），由平台发行和回收，避免价格波动影响业务。

2. **双通证分层**：FCX 作为价值媒介，用于支付、结算、兑换；FCX2 作为权益积分，记录贡献，不可交易，仅用于兑换平台内增值权益。

3. **正向激励**：通过 FCX2 奖励优质行为（低纠纷、高好评、快速响应），形成信誉与利益挂钩的自驱生态。

4. **闭环流通**：FCX 在平台内循环，通过手续费、提现费等机制回收，保持总量与业务规模匹配。

5. **合规优先**：不涉及区块链或加密货币，仅作为数据库积分，规避金融监管风险。

#### FCX 总量与发行机制

**发行方式**：

- 平台作为唯一发行方，用户通过法币购买 FCX（如 1 USD = 100 FCX）
- 初始发行量根据预估业务规模设定，后续按需增发
- 平台预留一部分 FCX 作为生态激励基金

**回收方式**：

- 用户提现时收取 3% 手续费，FCX 被销毁或回收
- 交易佣金、订阅费等最终以法币结算，对应 FCX 从流通中回收
- 用户使用 FCX 支付平台服务时，FCX 被平台回收

**价格稳定机制**：

- 平台始终以固定汇率出售 FCX，不设二级市场
- 提现时按固定汇率兑换，但收取手续费防止套利
