# 高管仪表板 - 快速启动指南

## 🚀 立即开始使用

### 1. 访问仪表板

**URL**: `https://your-domain.com/enterprise/admin/executive-dashboard`

**权限要求**:

- `enterprise_admin` (企业管理员)
- `executive_view` (高管查看权限)
- `bi_analytics` (BI 分析权限)

### 2. 首次使用

#### 步骤 1: 登录企业后台

```bash
# 打开浏览器访问
https://your-domain.com/login

# 输入管理员账号密码
```

#### 步骤 2: 导航到仪表板

- 点击左侧菜单「高管仪表板」
- 或直接在地址栏输入完整 URL

#### 步骤 3: 查看数据

系统会自动加载最新的经营数据，无需额外配置。

## 🎯 常用操作

### 筛选数据

#### 按时间范围

1. 点击右上角时间下拉框
2. 选择：最近 7 天 / 30 天 / 90 天 / 1 年
3. 数据自动更新

#### 按类别

1. 点击类别下拉框
2. 选择：全部 / 财务 / 用户 / 业务 / 运营
3. 只显示对应类别的 KPI

#### 搜索

1. 在搜索框输入关键词
2. 实时过滤告警列表

### 深度分析（钻取）

#### 步骤

1. **点击任意 KPI 卡片**
   - 例如：点击「月度收入」卡片

2. **查看深度分析弹窗**
   - 核心指标概览
   - 时间序列趋势
   - 维度分解分析
   - TOP 表现排行
   - AI 智能洞察

3. **切换分析维度**
   - 点击「趋势分析」Tab
   - 点击「维度分解」Tab
   - 点击「表现排行」Tab

4. **导出数据**
   - 点击右上角「导出」按钮
   - 选择 Excel 或 PDF 格式

### 处理告警

#### 查看告警

1. 点击右上角「告警中心」按钮
2. 查看告警列表

#### 处理告警

- **确认告警**: 点击「确认告警」按钮
- **解决告警**: 点击「标记解决」按钮
- **忽略告警**: 关闭对话框

#### 历史记录

1. 切换到「历史记录」Tab
2. 查看已解决的告警
3. 点击「导出历史记录」

## 🔧 故障排查

### 问题 1: 数据不更新

**解决方案**:

1. 检查网络连接
2. 手动点击刷新按钮
3. 清除浏览器缓存
4. 重新登录

### 问题 2: 无法访问仪表板

**解决方案**:

1. 确认账号权限
2. 检查 URL 是否正确
3. 联系管理员开通权限

### 问题 3: 移动端显示异常

**解决方案**:

1. 刷新页面
2. 旋转屏幕（横竖屏切换）
3. 更新浏览器到最新版本

### 问题 4: 离线模式不可用

**解决方案**:

1. 确保首次访问时有网络
2. 允许浏览器存储数据
3. 检查 Service Worker 是否注册成功
