## ✨ 特性说明

### 1. 完整的用户信息

每个用户都包含：

- ✅ 基本信息（邮箱、手机、头像）
- ✅ 状态信息（活跃/待审核/审核中）
- ✅ 认证信息（已认证/未认证/审核中）
- ✅ 订阅计划（免费/基础/专业/企业）
- ✅ 角色分配
- ✅ 时间戳（创建时间、更新时间）

### 2. 详细的扩展信息

#### 个人用户详情

- 姓名、昵称、性别、生日
- 地址信息（省市区）
- 会员等级和积分

#### 维修店详情

- 店铺名称、类型、营业执照
- 店铺描述、特色服务
- 联系信息、地址
- Logo、认证等级
- 营业时间、服务区域

#### 企业/外贸公司详情

- 公司名称、类型、税号
- 公司简介、行业、规模
- 业务范畴、主营产品
- 目标市场
- 联系方式、地址
- 网站、社交媒体

### 3. 真实的测试数据

- 🎨 **头像**: 使用 DiceBear API 生成随机头像
- 📧 **邮箱**: 统一使用 example.com 域名
- 📱 **手机**: 使用规范的虚拟号码
- 🏢 **地址**: 使用真实的中国城市地址格式
- 💼 **公司信息**: 符合中国公司命名规范

## ⚠️ 注意事项

### 1. 数据清理

如果需要删除所有示例数据：

```sql

WHERE email LIKE '%example.com';

-- 验证是否删除成功
SELECT COUNT(*) FROM user_accounts WHERE email LIKE '%example.com';

```
