## 🚀 技术架构增强

### 新增服务组件

```
src/fcx-system/
├── services/
│   ├── repair-order.service.ts     # 工单管理服务
│   ├── fcx2-option.service.ts      # FCX2期权服务
│   └── fcx2-reward.service.ts      # FCX2奖励服务
├── api/fcx/
│   ├── orders/                     # 工单API路由
│   │   ├── route.ts               # 工单列表和创建
│   │   └── [orderId]/route.ts     # 工单详情和操作
│   └── rewards/                   # 奖励API路由
│       └── route.ts               # 奖励计算和管理
```

### 核心功能API接口

- `POST /api/fcx/orders` - 创建维修工单
- `GET /api/fcx/orders` - 查询工单列表
- `GET /api/fcx/orders/[orderId]` - 获取工单详情
- `PUT /api/fcx/orders/[orderId]` - 工单操作（确认/完成/取消）
- `POST /api/fcx/rewards` - 奖励计算和发放
- `GET /api/fcx/rewards` - 奖励查询和统计
