## 功能描述

开发用户扫码后看到的H5页面，根据二维码ID展示对应产品的电子说明书、AI诊断入口和附近维修店推荐。

## 技术实现

### 核心功能模块

#### 1. 产品信息展示 ✅

- **实现方式**: 从`/api/products/{id}`获取产品信息
- **展示内容**:
  - 产品名称、品牌、型号、分类
  - 产品描述信息
  - 品牌Logo展示
- **技术特点**: 响应式设计，移动端优先

#### 2. 多语言说明书切换 ✅

- **支持语言**: 中文(zh)、英文(en)
- **实现方式**:
  - 语言切换按钮控制
  - 根据当前语言过滤可用说明书
  - 动态显示对应语言的说明书标题
- **用户体验**:
  - 清晰的语言标识
  - 实时切换无刷新
  - 无对应语言说明书时给出提示

#### 3. 设备自动识别 ✅

- **识别方式**: 基于User-Agent检测
- **支持设备类型**:
  - iOS设备(iPhone/iPad)
  - Android设备
  - Windows PC
  - macOS设备
- **置信度评估**: 不同设备类型给出相应置信度
- **手动选择**: 提供手动选择设备选项作为备选

#### 4. AI诊断功能跳转 ✅

- **跳转路径**: `/diagnosis/{productId}`
- **前提条件**: 必须先识别设备
- **按钮状态**:
  - 未识别设备时禁用
  - 识别设备后启用

#### 5. 附近维修店推荐 ✅

- **API接口**: `/api/shops/nearby`
- **定位方式**:
  - 优先使用浏览器地理位置API
  - 失败时使用默认位置(北京)
- **推荐算法**: 基于Haversine公式的距离计算
- **展示信息**:
  - 店铺名称、联系方式
  - 距离、评分、评价数
  - 服务项目、认证状态
  - 地址信息

### 技术架构

#### 前端技术栈

- **框架**: Next.js 14.0.0
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **组件**: React Hooks (useState, useEffect)
- **路由**: App Router

#### 后端API

- **附近店铺API**: `/api/shops/nearby`
  - GET参数: `lat`, `lng`, `radius`, `limit`
  - 支持POST复杂查询
  - 距离计算使用Haversine公式
- **产品信息API**: `/api/products/{id}`
- **诊断跳转**: `/diagnosis/{id}`
