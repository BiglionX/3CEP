## 🚀 使用示例

### 快速开始

```bash
# 启动演示
npm run realtime:demonstrate

# 性能测试
npm run realtime:test-performance

# 实时监控
npm run realtime:monitor
```

### 代码示例

```javascript
// 触发价格更新事件
await businessEventTrigger.triggerPriceUpdate({
  partId: 'PART001',
  oldPrice: 100.0,
  newPrice: 85.0,
  changePercent: -15.0,
  platform: 'taobao',
});

// 配置告警规则
monitoringAlertSystem.addBacklogAlert({
  topic: 'price_update',
  threshold: 1000,
  checkInterval: 30000,
  alertChannels: ['email', 'slack'],
});
```
