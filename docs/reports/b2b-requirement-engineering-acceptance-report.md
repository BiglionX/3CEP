## 技术架构

### 核心组件

- **需求理解服务** (`requirement-understanding.service.ts`) - 主协调器
- **输入检测器** (`input-detector.service.ts`) - 输入类型识别和路由
- **图像处理器** (`image-processor.service.ts`) - OCR 和图像分析
- **链接提取器** (`link-extractor.service.ts`) - 网页内容抓取
- **大模型解析器** (`large-model-parser.service.ts`) - AI 需求解析

### 数据模型扩展

- 新增 `InputType` 枚举支持多模态输入
- 扩展 `RawProcurementRequest` 和 `ParsedProcurementRequest` 接口
- 添加图片链接、源链接、提取内容等字段

### API 接口

- **端点**: `/api/b2b-procurement/parse-requirement`
- **方法**: POST (处理请求), GET (健康检查)
- **输入**: 支持自动检测或指定输入类型
- **输出**: 标准化的结构化采购需求
