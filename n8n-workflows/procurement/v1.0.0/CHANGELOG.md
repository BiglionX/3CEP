# 采购工作流 v1.0.0 变更日志

## 发布日期
2026-02-20

## 版本摘要
初始版本发布，包含基础采购需求解析和高级错误处理功能。

## 新增功能
- ✅ 基础采购需求解析工作流 (`b2b-procurement-basic.json`)
  - 简单的Webhook到API调用流程
  - 基础的请求参数处理
  - 标准化响应格式输出

- ✅ 高级采购需求解析工作流 (`b2b-procurement-advanced.json`)
  - 完整的错误处理机制
  - 请求元数据和日志记录
  - 条件分支和复杂响应处理
  - 置信度评分和处理时间统计

## 技术特性
- 支持n8n 0.200.0及以上版本
- 集成B2B采购需求解析API
- 标准化的JSON响应格式
- 完善的错误处理和日志记录

## 配置要求
- 需要配置`baseUrl`参数指向B2B解析API
- 支持环境变量配置
- Webhook路径: `/webhook/b2b-procurement-parse` 和 `/webhook/b2b-procurement-advanced`
