# 行为准则

## 🎯 我们的承诺

为了营造开放、包容、友善的社区环境，我们作为 FixCycle 项目的维护者和参与者，承诺使参与我们项目的每个人都能够不受骚扰地参与，无论其经验水平、性别、性别认同和表达、性取向、残疾、外貌、体型、种族、民族、年龄、宗教或国籍。

## 🤝 我们的标准

### 积极的行为包括：

- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性的批评
- 关注对社区最有利的事情
- 对其他社区成员表现出同理心

### 不可接受的行为包括：

- 使用性化的语言或图像，以及不受欢迎的性关注或挑逗
- 恐吓、骚扰、侮辱或贬低性的评论和人身攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息，如物理或电子地址
- 其他可能被认为是不适当或违反职业操守的行为

## 🔧 执行责任

项目维护者有责任澄清可接受的行为标准，并对任何不可接受的行为采取适当和公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、维基编辑、问题和其他贡献，或暂时或永久禁止任何他们认为行为不当、威胁、冒犯或有害的贡献者。

## 🌍 适用范围

本行为准则适用于项目空间内，也适用于在公共场所代表项目的情况。代表项目的情形包括使用官方项目电子邮件地址、通过官方社交媒体账号发帖，或在在线或线下活动中担任指定代表。

## 📋 执行指南

项目维护者将遵循这些社区影响指南来确定对其认为违反本行为准则的任何行为的后果：

### 1. 纠正

**社区影响**：使用不当的语言或其他在社区中被认为不专业的行为。

**后果**：项目维护者的私下书面警告，明确说明违规行为的性质，并解释为什么该行为不适当。可能会要求为此行为公开道歉。

### 2. 警告

**社区影响**：单一事件或一系列行为中的违规行为。

**后果**：警告及后果（如继续不当行为可能导致临时或永久禁止）。

### 3. 临时禁止

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定时间内禁止与社区进行任何形式的互动或公开交流。在此期间，不得代表该项目进行公开或私下互动，包括外部渠道如社交媒体。违反这些条款可能导致永久禁止。

### 4. 永久禁止

**社区影响**：表现出违反社区标准的行为模式，包括持续的不当行为、骚扰个人或攻击或贬低某个群体。

**后果**：永久禁止在社区内进行任何形式的公开互动。
