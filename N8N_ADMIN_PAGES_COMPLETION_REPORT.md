## 🚀 使用方式

### 1. 访问页面

确保已登录 Admin 后台，访问对应路由即可。

### 2. 环境变量配置

在 `.env` 文件中配置：

```bash
NEXT_PUBLIC_N8N_URL=http://localhost:5678
N8N_API_TOKEN=your-api-token
```

### 3. API 连接

所有页面会自动尝试连接真实的 n8n API，如果连接失败会显示模拟数据用于演示。
