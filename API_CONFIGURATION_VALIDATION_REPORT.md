## 🚀 使用指南

### 访问 API 配置管理

1. 登录管理后台
2. 在左侧导航菜单中点击"API 配置"
3. 进入 API 配置管理页面

### 配置 API 密钥

1. 选择对应的 API 分类标签页
2. 找到需要配置的 API 服务
3. 输入 API 密钥或连接信息
4. 点击"保存"按钮
5. 可选择"测试所有 API"验证连接

### 监控配置状态

- 查看顶部状态概览卡片
- 关注必需配置的完成情况
- 定期运行连接测试确保 API 可用性
