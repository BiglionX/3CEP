## 🚀 使用指南

### 访问路径
```
智能体管理页面: /agents
Playground 环境: 通过智能体列表中的 "Playground" 按钮访问
```

### 基本操作流程
1. **创建智能体**: 点击 "新建智能体" 按钮配置基本信息
2. **激活智能体**: 将智能体状态设置为 "active"
3. **在线测试**: 在 Playground 中输入测试内容并执行
4. **调试分析**: 使用调试模式查看详细的执行过程
5. **查看历史**: 在执行记录中查看历史执行结果

### API 调用示例
```bash
# 获取智能体列表
curl -X GET http://localhost:3001/api/agents

# 执行智能体
curl -X POST http://localhost:3001/api/agents/{id}/execute \
  -H "Content-Type: application/json" \
  -d '{"input_data": {"query": "测试问题"}}'

# 调试执行
curl -X PATCH http://localhost:3001/api/agents/{id}/execute \
  -H "Content-Type: application/json" \
  -d '{"input_data": {"query": "调试问题"}, "parameters": {"debug_mode": true}}'
```
