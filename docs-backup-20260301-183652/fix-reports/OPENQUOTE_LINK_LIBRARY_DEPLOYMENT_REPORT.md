## 🎯 使用说明

### 1. 访问管理后台

```
URL: http://localhost:3001/admin/links/library
功能: 链接优先级管理、状态监控、批量操作
```

### 2. API调用示例

```bash
# 查询链接
curl -X POST http://localhost:3001/api/links/query \
  -H "Content-Type: application/json" \
  -d '{"query":"iPhone电池更换","limit":3}'

# 调整优先级
curl -X PUT http://localhost:3001/api/links/priority \
  -H "Content-Type: application/json" \
  -d '{"updates":[{"id":"链接ID","priority":80}]}'
```
