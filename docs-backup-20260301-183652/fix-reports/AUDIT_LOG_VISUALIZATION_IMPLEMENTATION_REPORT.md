## 🚀 使用指南

### 访问路径

```
审计日志页面: /audit
```

### 主要功能操作

1. **日志浏览**: 查看系统操作日志和详细信息
2. **数据筛选**: 按操作类型、时间范围、严重程度等条件筛选
3. **统计分析**: 查看操作分布、趋势分析等统计图表
4. **日志导出**: 导出符合条件的日志数据（JSON/CSV格式）
5. **异常监控**: 识别和跟踪异常行为模式

### API 调用示例

```bash
# 获取审计日志
curl -X GET "http://localhost:3001/api/audit/logs?action=login&severity=warning"

# 导出审计日志
curl -X POST http://localhost:3001/api/audit/logs \
  -H "Content-Type: application/json" \
  -d '{"format": "csv", "filters": {"action": "create"}}'
```
