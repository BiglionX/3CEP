## 🚀 使用指南

### 管理员操作步骤

1. **访问配置管理**
   - 登录管理后台
   - 导航至"API配置"页面

2. **配置供应链服务**
   - 在"供应链管理"分类下配置相关API
   - 填写服务URL和认证信息
   - 点击"测试连接"验证配置

3. **配置智能服务**
   - 在"AI服务"和"采购代理"分类下配置
   - Weaviate向量数据库API配置
   - 采购代理服务API配置

4. **配置基础设施**
   - 在"消息服务"配置邮件服务
   - 在"监控服务"配置系统监控API

### 验证配置有效性

- 使用"测试所有API"功能批量验证
- 查看配置状态统计卡片
- 监控API健康度变化
