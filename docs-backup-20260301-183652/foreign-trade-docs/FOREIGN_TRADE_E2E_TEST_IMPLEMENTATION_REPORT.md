## 🎯 测试目标

实现完整的外贸采购业务流程端到端测试，覆盖以下核心业务场景：

1. **A外贸公司**创建采购订单
2. **采购智能体**自动分解订单并向1-3家历史供应商询价
3. **供应商外贸公司**接收订单，销售智能体自动生成报价
4. **供应商**可选择修改报价后提交
5. **A采购智能体**接收报价后自动加毛利
6. **A人工审核**后发送给客户（消息或邮件）
7. **客户**收到报价后确认下单，进入正式订单流程
