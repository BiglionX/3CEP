## 🛡️ 安全特性实现

### 多层级限流策略

- **API级别**: 通用API接口限流 (1000 req/min)
- **敏感操作**: 高风险操作限流 (15-20 req/min)
- **认证相关**: 登录注册等限流 (5 req/min)
- **搜索功能**: 搜索接口限流 (30 req/min)

### 智能保护机制

- **自适应匹配**: 根据请求路径和方法自动选择限流规则
- **封禁机制**: 超限后临时封禁客户端防止恶意攻击
- **管理员豁免**: 为管理员提供更宽松的访问权限
- **详细日志**: 记录限流事件便于监控和分析
