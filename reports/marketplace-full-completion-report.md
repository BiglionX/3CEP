## 🏗️ 系统架构总览

### 完整目录结构

```
src/app/marketplace/
├── page.tsx                    # 市场主页
├── [id]/page.tsx              # 智能体详情页
├── categories/[category]/     # 分类浏览页
├── cart/page.tsx              # 购物车页面
├── checkout/page.tsx          # 结账页面
├── orders/page.tsx            # 订单管理页面
├── tokens/page.tsx            # Token钱包页面
└── developer/page.tsx         # 开发者收益页面
└── enterprise/page.tsx        # 企业订阅页面

src/app/api/marketplace/
├── route.ts                   # 市场基础API
├── orders/route.ts           # 订单管理API
├── tokens/route.ts           # Token经济API
└── enterprise/route.ts       # 企业订阅API
```

### 技术栈汇总

- **前端框架**: Next.js 14 (App Router)
- **编程语言**: TypeScript
- **UI组件库**: shadcn/ui + Tailwind CSS
- **状态管理**: React Hooks
- **后端服务**: Next.js API Routes
- **数据库**: Supabase (PostgreSQL)
- **图标库**: Lucide React
